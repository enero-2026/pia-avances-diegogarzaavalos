# @balancehogar/constants

Constantes del dominio. Vive aparte de `utils` porque son **datos**, no funciones, y se usan tanto para validación (en `schemas`) como para UI.

## Contenido previsto

- `categories` — servicios, despensa, transporte, salud, otros (+ ícono y color por categoría).
- `colors` — paleta para diferenciar miembros visualmente.
- `recurrence` — periodicidades de gastos recurrentes.
- `status` — estados (`pending`, `paid`) para gastos recurrentes.
